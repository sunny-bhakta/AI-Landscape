
# PyTorch Overview

**PyTorch** is a Python-based scientific computing package widely used for deep learning and machine learning. It serves two main purposes:

- **Replacement for NumPy:** Leverages the power of GPUs and other accelerators for fast computation.
- **Automatic Differentiation:** Provides tools for building and training neural networks with ease.

## Key Features

- Flexible and powerful deep learning library
- Comprehensive set of tools for building, training, and deploying models
- Strong support for research and production

## Core Concept: Tensors

- **Tensors:** N-dimensional arrays that are the fundamental data structure in PyTorch.
	- Support automatic differentiation
	- Enable hardware acceleration (GPU/TPU)
	- Provide a rich API for mathematical operations
